Introduction

Node.js

Node.js is a cross-platform, open-source JavaScript runtime environment that can run on Windows, Linux, Unix, macOS, and more. Node.js runs on the V8 JavaScript engine, and executes JavaScript code outside a web browser.

Why Node.js?

Node.js excels at handling many simultaneous connections with minimal overhead, making it perfect for:
- Real-time applications (chats, gaming, collaboration tools)
- APIs and microservices
- Data streaming applications
- Command-line tools
- Server-side web applications
Its non-blocking, event-driven architecture means it can handle many operations at the same time without waiting for each one to finish..

JavaScript and Node

JavaScript is a programming language. main.js would be a file containing JavaScript. Node.js is a runtime for actually executing your code. You give it JavaScript in the form of a file, and it executes the instructions in your code outside of the browser. You would execute a JavaScript file called main.js with the command node main.js after navigating to the location of the file in your terminal.

Node.js and JavaScript together allow us to build both frontend and backend applications using the same language.

REPL

A REPL (Read-Eval-Print Loop) is an interactive computer programming environment that takes single user inputs, executes (evaluates) them, and immediately returns the result. Commonly used for testing, debugging, and learning languages like Python or JavaScript, it provides instant feedback via a console/terminal without needing to run a full file.
